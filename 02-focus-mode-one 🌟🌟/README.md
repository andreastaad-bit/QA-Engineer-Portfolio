**Focus Mode On** is a productivity web application I designed and built to help people reach a state of deep focus while studying or working.

The idea came from a personal challenge. I get distracted easily, and although there are many focus apps available, most require a paid subscription or include advertisements that interrupt concentration. Even YouTube study videos eventually become distracting because of ads or recommendations.

Instead of relying on existing solutions, I decided to build my own.

The goal was to create a clean, distraction-free environment that I could use every day while also serving as a personal project to practice software development and quality assurance.

🌐 **Live Website**

https://focusmodeon.lat/

---

## The Problem

Maintaining concentration for long periods is difficult.

Common productivity tools often have limitations such as:

* Subscription fees
* Advertisements
* External distractions
* Lack of scheduled movement reminders
* No breathing exercises to help regain focus

I wanted a single place where I could simply start working without interruptions.

---

## The Solution

Focus Mode On combines several productivity techniques into one simple web application.

The application includes:

* Pomodoro-style focus sessions
* Break reminders
* Stretch reminders
* Box breathing exercises
* Calm background music
* A clean and distraction-free interface

The objective is not only to improve productivity but also to encourage healthy work habits during long study sessions.
